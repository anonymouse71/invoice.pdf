# invoice.pdf
Invoice PDF Generator using dompdf

## Usage

1. ```git clone https://github.com/sohelamin/invoice.pdf.git```
2. ```cd invoice.pdf```
3. ```composer install```
4. Visit http://localhost/invoice.pdf/
